¿Qué es JSON?

JSON significa Notación de Objetos de JavaScript. Es una forma de almacenar y organizar los datos que se puede leer tanto por las máquinas como por los humanos.

¿Cómo se ve el JSON?

Un documento JSON se compone de pares de claves y valores. Aquí tienes un ejemplo muy sencillo: 

                {
                    "nombre": "Juan",
                    "edad": 30,
                    "ciudad": "Madrid"
                }

En este ejemplo, "nombre", "edad" y "ciudad" son las claves, y "Juan", 30 y "Madrid" son los valores correspondientes.

¿Cómo se usa JSON?

JSON se utiliza para almacenar y transportar datos. Por ejemplo, si tienes una aplicación que necesita guardar los datos del usuario, podrías usar JSON para hacerlo. Cuando necesites recuperar esos datos, simplemente lees el archivo JSON.

JSON también se utiliza para enviar datos a través de Internet. Por ejemplo, si tienes una aplicación que necesita obtener datos del tiempo de un servicio web, ese servicio podría enviarte los datos en formato JSON.
